# Python - if/else, loops, functions

Project covering Python control flow: conditional statements (`if`/`elif`/`else`),
loops (`for`/`while`), and defining functions. Several tasks restrict which
built-ins may be used to force practice with `ord()`, slicing, and string
formatting.

## Tasks

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, zero, or negative |
| `1-last_digit.py` | Prints and classifies the last digit of a random number |
| `2-print_alphabet.py` | Prints the lowercase ASCII alphabet |
| `3-print_alphabt.py` | Prints the lowercase alphabet except `q` and `e` |
| `4-print_hexa.py` | Prints 0–98 in decimal and hexadecimal |
| `5-print_comb2.py` | Prints numbers 0–99, comma-separated |
| `6-print_comb3.py` | Prints all distinct two-digit combinations |
| `7-islower.py` | Function: checks for a lowercase character |
| `8-uppercase.py` | Function: prints a string in uppercase |
| `9-print_last_digit.py` | Function: prints and returns the last digit of a number |
| `10-add.py` | Function: adds two integers |
| `11-pow.py` | Function: computes a to the power of b |
| `12-fizzbuzz.py` | Function: prints FizzBuzz from 1 to 100 |
