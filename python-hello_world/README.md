# Python - Hello, World

Introductory project to the Python programming language. It covers running
Python from shell scripts, printing text, working with strings (concatenation,
slicing, and formatting with f-strings), and basic Python tooling.

## Tasks

| File | Description |
|------|-------------|
| `0-run` | Shell script that runs a Python file named in `$PYFILE` |
| `1-run_inline` | Shell script that runs Python code stored in `$PYCODE` |
| `2-print.py` | Prints a fixed string using `print` |
| `3-print_number.py` | Prints an integer with an f-string (no casting) |
| `4-print_float.py` | Prints a float with 2-digit precision via an f-string |
| `5-print_string.py` | Prints a string 3 times then its first 9 characters |
| `6-concat.py` | Concatenates two strings into a greeting |
| `7-edges.py` | Slices a word into first 3, last 2, and middle parts |
| `8-concat_edges.py` | Builds a sentence using only slicing (no literals) |
| `9-easter_egg.py` | Prints "The Zen of Python" |
