# Python - import & modules

Project on importing modules and functions in Python, command-line arguments
via `sys.argv`, inspecting modules with `dir()`, and guarding code with
`if __name__ == "__main__"` so it does not run on import.

## Tasks

| File | Description |
|------|-------------|
| `0-add.py` | Imports `add` from `add_0.py` and prints `1 + 2 = 3` |
| `1-calculation.py` | Imports add/sub/mul/div from `calculator_1.py` and prints results |
| `2-args.py` | Prints the number and list of command-line arguments |
| `3-infinite_add.py` | Prints the sum of all command-line arguments |
| `4-hidden_discovery.py` | Prints public names defined in `hidden_4.pyc` |
| `5-variable_load.py` | Imports the variable `a` from `variable_load_5.py` and prints it |

The helper modules (`add_0.py`, `calculator_1.py`, `variable_load_5.py`) and
the compiled `hidden_4.pyc` are provided by the checker at run time.
